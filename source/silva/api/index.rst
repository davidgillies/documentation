Silva Content API
=================

This present the Silva API. It is mostly created from Silva
interfaces.

.. glossary::

   *Zope interface*
     An interface is a Zope concept used to describe what an object
     (or a class) provides (or implements) as methods and attributes,
     and what are their signatures (argument list).
